# PHASE 5 — CREATE FINANCIAL VIEWS

Important for performance.

Create SQL views.

---

## View 1 — Monthly Category Summary

CREATE VIEW monthly_category_summary AS

SELECT
profile_id,
category_id,
DATE_TRUNC('month', due_date) as month,
SUM(amount) as total

FROM transactions

GROUP BY
profile_id,
category_id,
month;

---

## View 2 — Credit Card Summary

CREATE VIEW credit_card_summary AS

SELECT
credit_card_id,
DATE_TRUNC('month', due_date) as month,
SUM(amount) as total

FROM transactions

GROUP BY
credit_card_id,
month;

# PHASE 6 — CREATE KNOWN QUERY INTENTS

Create:

src/ai/known-intents.ts

Purpose:

Speed up common financial questions.

---

## Examples

Saldo Conta:

SELECT balance
FROM bank_accounts
WHERE name ILIKE '%{accountName}%'

---

Gasto Categoria:

SELECT SUM(amount)
FROM transactions
JOIN categories ON categories.id = transactions.category_id
WHERE categories.name = '{categoryName}'
AND DATE_TRUNC('month', due_date) = '{month}'

---

Gasto Cartão:

SELECT SUM(amount)
FROM transactions
WHERE credit_card_id = '{cardId}'
