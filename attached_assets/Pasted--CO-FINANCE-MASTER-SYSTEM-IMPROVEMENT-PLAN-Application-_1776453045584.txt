# CO-FINANCE — MASTER SYSTEM IMPROVEMENT PLAN

## Application Name

Co-Finance

---

# SYSTEM ARCHITECTURE OVERVIEW

The system supports:

User Account → Multiple Profiles

Each user account can contain multiple financial profiles.

Each profile contains:

* Transactions
* Accounts Payable
* Accounts Receivable
* Credit Cards
* Installments
* Financial reports

Example:

Conta do Usuário:

* Perfil Pessoal
* Perfil Empresa
* Perfil Familiar

All financial data belongs to a Profile.
